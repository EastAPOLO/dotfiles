# Dependencies

If you decide to copy and paste the configuration files, install the following before doing so:

My browser in the config

[firefox](https://www.mozilla.org/en-US/firefox/new/)

Software launcher

[rofi](https://davedavenport.github.io/rofi/)

Status command

[i3blocks](https://github.com/vivien/i3blocks)

Screenshot tool

[maim](https://github.com/naelstrof/maim)

Terminal emulator

[termite](https://github.com/thestinger/termite/)

Compositing manager

[compton](https://github.com/chjj/compton)

Wallpaper setter that is also an awesome image viewer

[feh](https://github.com/derf/feh)

No gaps No go

[i3-gaps](https://github.com/Airblader/i3)

Font Awesome, a full suite of 675 pictographic icons for easy scalable vector graphics on websites, created and maintained by Dave Gandy. 

[font-awesome](https://github.com/FortAwesome/Font-Awesome)
